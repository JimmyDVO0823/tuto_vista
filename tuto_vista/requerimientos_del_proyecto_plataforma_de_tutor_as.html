# Proyecto Final: Plataforma de Tutorías Académicas ## Contexto Arquitectura cliente-servidor con Frontend en React.js y Tailwind CSS, consumiendo una API REST. ## 1. Vistas Generales (Públicas) - **Landing Page:** Presentación de la plataforma. - **Login y Registro:** Formularios con validación y sistema de autenticación. ## 2. Rol: Estudiante - **Dashboard:** Próximas tutorías, notificaciones y resumen. - **Buscador de Tutorías:** Filtros por materia, precio y calificación. - **Perfil del Tutor (Detalle):** Información, especialidad y calendario de reserva. - **Mis Tutorías (Historial):** Tabla con historial y acciones (cancelar/reseñar). ## 3. Rol: Tutor - **Dashboard:** Estadísticas (horas, calificación) y agenda del día. - **Gestión de Disponibilidad:** CRUD de bloques horarios. - **Mis Alumnos / Solicitudes:** Gestión de reservas (aceptar/rechazar). - **Perfil Público (Edición):** Formulario con subida de imagen y bio. ## 4. Rol: Administrador - **Dashboard Administrativo:** Reportes avanzados y gráficos globales. - **Gestión de Usuarios:** Tabla maestra de control. - **Gestión de Materias:** CRUD de asignaturas oficiales. ## Requisitos Técnicos - Diseño Responsive. - Código preparado para API (Mock data + Comentarios TODO). - Componentización lógica y limpia. - Estética moderna y académica.
